The folder "mini_lagnuage_compiler" represents a package inside which Java files are located.  (interpreter would be a more fitting name)
The submitted solution should be platform-independent.
The submitted solution accepts a single program argument through which a file path (of the program file to be interpreted) should be specified.
Inside the test folder are some program files, of which only the first one is correctly formed. The rest of them are ill-formed and serve to demonstrate
submitted solutions error reporting capabilities.
